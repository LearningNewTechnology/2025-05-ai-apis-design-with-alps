Task Management API Profile

This API manages task records including creation, editing, filtering, and assignment actions.

Data Descriptors

ID Title Definition
id Identifier A globally unique identifier for each task record.
title Title Short title or name of the task.
description Description Detailed description of the task.
dueDate Due Date Deadline by which the task should be completed.
status Status Current state of the task (active or completed).
priority Priority Task importance, typically from 1 (lowest) to 5 (highest).
assignedUser Assigned User User responsible for completing the task.
